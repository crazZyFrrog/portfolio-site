# Портфолио — Владислав Левоненко

Сайт-портфолио вайбкодера: кейсы salonlt.ru и конструктора лендингов.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## Структура

- `/` — главная (о себе, кейсы, контакт)
- `/cases/salon-lt` — кейс салона красоты
- `/cases/landing-constructor` — кейс MVP конструктора

Контент: `src/data/profile.ts`, `src/data/cases.ts`
